# Encrt — Encurtador de Links

Encurtador de URLs minimalista desenvolvido com **Node.js** e **PostgreSQL**. O projeto permite transformar links longos em URLs curtas com contagem de cliques, geração automática de **QR Code** e histórico de navegação salvo localmente.

---

## Tecnologias Utilizadas

- **Backend:** Node.js e Express.js
- **Banco de Dados:** PostgreSQL
- **Frontend:** HTML5, CSS3 e JavaScript
- **Funcionalidades:** Geração de QR Code e Histórico via LocalStorage
---

## Como rodar o projeto

### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado
- [PostgreSQL](https://www.postgresql.org/) rodando localmente

### 1. Instalação
```bash
# Clone o repositório
git clone [https://github.com/adriiana-dev/encrt.git](https://github.com/adriiana-dev/encrt.git)

# Entre na pasta
cd encrt

# Instale as dependências
npm install
